# 🏠 물리학습 지원 포털 (Physics Learning Portal)

이 프로젝트는 학생들이 물리학의 핵심 개념을 직관적으로 이해할 수 있도록 돕기 위해 제작된 대화형 시뮬레이션 포털입니다.

## 📍 주요 기능
- **위치 벡터와 변위**: 평면 위에서의 위치 벡터와 변위를 시각화하고 계산합니다.
- **평균 속도 탐구**: 운동 경로를 따라 변위와 평균 속도 벡터의 관계를 탐구합니다.
- **가속도와 포물선 운동**: 포물선 운동 궤적에서 속도 변화를 통해 가속도의 방향을 탐구합니다.

## 🚀 시작하기

### 1단계: 저장소 복제 (Clone)
```bash
git clone https://github.com/yuno0154/physics-course.git
cd physics-course
```

### 2단계: 가상환경 설정 및 실행
가상환경이 설치되지 않은 경우, 포함된 설정 스크립트를 사용하여 자동으로 설치하고 실행할 수 있습니다.

- **Windows**: `run_portal.bat` 실행
- **수동 설치**:
  ```bash
  python -m venv .venv
  source .venv/Scripts/activate  # Windows
  pip install -r requirements.txt
  streamlit run main_app.py
  ```

## 🛠 기술 스택
- **Python**: 3.10+
- **Streamlit**: 웹 인터페이스 구축
- **Plotly**: 대화형 데이터 시각화
- **NumPy**: 물리 계산 수행
